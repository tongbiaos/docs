### GifCam

### 去除衣服油渍

吃火锅时，白色衣服上面粘了油渍，使用洗衣液、洗衣粉、肥皂、牙膏等直接搓洗，效果不明显，记录一下其他方法。

洗衣液直接涂抹，静置一段时间，【实测无效，是不是可能和油污的种类有关】
